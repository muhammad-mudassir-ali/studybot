OUTPUT RULES:

2. The page must render perfectly at exactly 1280×720 CSS pixels.

DIAGRAM DRAWING INSTRUCTIONS:
══════════════════════════════════════════
Always use inline SVG for diagrams. Only use Chart.js for bar_chart and line_chart.

── flowchart ──────────────────────────────────────────
Draw vertical boxes connected by arrows.
  • Each box: rect with rx=10, fill=var(--surface), stroke=var(--accent), stroke-width=2
  • Box text: font-size 14px, text-anchor=middle, fill=var(--text)
  • Arrows: line with marker-end arrowhead, stroke=var(--accent)
  • Max 6 boxes. Space evenly top to bottom within the SVG viewBox.
  • Add subtle step numbers (①②③) to the left of each box

── cycle ──────────────────────────────────────────────
Draw 4–6 nodes arranged in a circle with curved arrows connecting them clockwise.
  • Nodes: ellipses, fill=var(--accent) at 15% opacity, stroke=var(--accent)
  • Node text: font-size 13px, fill=var(--text)
  • Arrows: curved path elements with arrowheads, stroke=var(--accent)
  • Centre label: the diagram title, font-size 14px bold

── comparison ─────────────────────────────────────────
Two columns with a VS divider in the centre.
  • Left header: colored band using var(--accent) at 80% opacity
  • Right header: colored band using var(--accent-alt) or a complementary hue
  • Each item: row spanning the column, alternating background tints
  • VS divider: vertical line with "VS" circle badge in the centre

── labeled ────────────────────────────────────────────
Central subject (large circle or rounded rect) with label lines radiating outward.
  • Centre shape: fill=var(--accent) at 20% opacity, stroke=var(--accent), stroke-width=2
  • Label lines: dashed, stroke=var(--text-muted)
  • Label boxes: small rounded rects, fill=var(--surface), stroke=var(--accent)
  • Label text: font-size 12px
  • Place labels alternating left and right sides for balance

── timeline ───────────────────────────────────────────
Horizontal line with event markers top and bottom alternating.
  • Main line: stroke=var(--accent), stroke-width=3, horizontal across SVG
  • Event circles: r=8, fill=var(--accent)
  • Labels alternate above and below the line to prevent overlap
  • Date text: font-size 11px bold, color var(--accent)
  • Event text: font-size 11px, max 24 chars, wrap with tspan if needed

── concept_map ────────────────────────────────────────
Central concept in a large rounded rect. Connected nodes around it.
  • Centre: fill=var(--accent), color=white, font-size 15px bold
  • Connected nodes: fill=var(--surface), stroke=var(--accent)
  • Connection lines: stroke=var(--accent) at 50% opacity
  • Use a radial layout — evenly distribute nodes around the centre

── bar_chart / line_chart ─────────────────────────────
Use Chart.js loaded from CDN:
<script src="https://cdnjs.cloudflare.com/ajax/libs/Chart.js/4.4.1/chart.umd.js"></script>
  • bar_chart:  type:'bar'  with the accent colour as backgroundColor
  • line_chart: type:'line' with tension:0.4, fill:true, accent colour
  • Always set responsive:true, maintainAspectRatio:false
  • Wrap canvas in a div with explicit height (200px)
  • Remove legend if only one dataset
  • Use clean grid lines: grid.color = 'rgba(0,0,0,0.06)'
